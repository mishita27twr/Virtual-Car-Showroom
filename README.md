# 🚗 AutoLux - Virtual Car Showroom

A modern, responsive **Virtual Car Showroom** website built using **HTML, CSS, and JavaScript**. The project allows users to explore luxury cars, view specifications, search for vehicles, and experience a premium online showroom interface.
